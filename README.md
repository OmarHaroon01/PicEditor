# PicEditor
Welcome to PicEditor
## Filters Available
* Blur
* Sketch
* Edge Detection
* Photocopy
* Erosion
* Dilation
* Grayscale
* Mirror
* Negative
* Vignetting
* Sepia
* Night Vision
* Histogram Equalization
* Posterize
* Contrast Highlighting

You can also compare your edited image with the original one. You can also save the image.
